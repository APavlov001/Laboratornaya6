## Создание репозитория GitHub
1.  Для начала я открываю папку с файлами который я хочу добавить в репозиторий GitHub в Visual Studio.
1.  Далее я выбираю в строке меню пункт git и в открывшемся списке выбираю пункт Создать репозиторий Git как показано на изображении 1.
![здесь должно было быть изображение 1](https://raw.githubusercontent.com/APavlov001/Laboratornaya6/master/png1.png "изображение 1")
1.  После открывается окно показанное на изображении 2 в котором можно выбрать создавать ли новый репозиторий на GitHub или Azure DevOps или создать его только локально, также в этом окне можно выбрать шаблон для файла .gitignore, этот файл отвечает какие файлы будут помечены как игнорируемые для git, также можно выбрать учетную запись GitHub в которой создастся репозиторий, указать имя репозитория и описание, также можно поставить галочку над пунктом Частный репозиторий, если галочка на этом пункте стоит то просматривать этот репозиторий смогут только некторые пользователи, если не ставить галочку то этот репозиторий смогут  просматривать все. После введения необходимых данных в этом окне я нажимаю кнопку Создать и отправить.

## Создание репозитория Azure DevOps

1. Для этого я открыл другую папку в которой нет git репозитория в Visual studio.

1. Далее я выбираю в строке меню пункт git и в открывшемся списке выбираю пункт Создать репозиторий Git как показано на изображении 4.
![здесь должно было быть изображение 4](https://raw.githubusercontent.com/APavlov001/Laboratornaya6/master/png4.png "изображение 4")
1. После открывается окно показанное на изображении 5 но заполнить просто так все поля в нем не выйдет. Поля организация и проект будут пустыми если вы не создавали в учетной записи Microsoft организацию и в этой ораганизации не создавали проект.
![здесь должно было быть изображение 5](https://raw.githubusercontent.com/APavlov001/Laboratornaya6/master/png5.png "изображение 5")

![здесь должно было быть изображение 6](https://raw.githubusercontent.com/APavlov001/Laboratornaya6/master/png6.png "изображение 6")
