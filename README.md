## Клонирование репозитория GitHub

1. Для этого я открыл другую папку в которой нет git репозитория в Visual studio.

1. Далее я выбираю в строке меню пункт git и в открывшемся списке выбираю пункт Клонирование репозиторий как показано на изображении 8 или 4 или 1.
![здесь должно было быть изображение 8](https://raw.githubusercontent.com/APavlov001/Laboratornaya6/master/png1.png "изображение 8")
1. После открывается окно показанное на изображении 9 в котором необходимо указать адрес репозитория и папку где он будет хранится.
![здесь должно было быть изображение 9](https://raw.githubusercontent.com/APavlov001/Laboratornaya6/master/png9.png "изображение 9")
Для узнавания адреса репозитория на GitHub необходимо зайти на интересующий вас репозиторий, потом справа сверху над списком файлов репозитория буде зеленая кнопка Code, нажимаете на нее и появитя список в котором вам необходимо в пункте clone (самый первый пункт) выбрать HTTPS и под этим пунктом будет отображен адрес репозитория как показано на изображении 10. Есть еще и более легкий метод для него нужно просто скопировать адрес начальной страницы репозитория и добавить в конце .git.

Также если вы вошли в github аккаунт в Visual Studio то вы можете выбрать githab в разделе посмотреть репозитории.
